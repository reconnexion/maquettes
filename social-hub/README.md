# Maquette de l'application "Social Hub"

Maquette imaginée lors du hackathon du 4 juin 2026 organisé par Reconnexion à l'Aubier

## Prompt passé à Claude

Dessine moi une application mobile qui permet à un individu ou à une organisation :
- de centraliser ses interactions, par exemple réponses aux commentaires, avec les utilisateurs des différents réseaux sociaux ;
- de pousser des contenus textuels, visuels, audios et audiovisuels sur nos différents réseaux sociaux en fonction du type de contenu ;
- d'agréger toutes les notifications provenant de l'ensemble de nos réseaux sociaux ;

## Réponses aux questions additionelles de Claude

- design_context: Non, propose une direction visuelle moderne
- networks: Instagram, Facebook, X (Twitter), TikTok, YouTube, LinkedIn, WhatsApp, Peertube, Mastodon, Pixelfed, Bluesky
- audience: Décide pour moi
- scope: Composer / publier du contenu, Tableau de bord d'accueil, Boîte de réception unifiée (interactions), Centre de notifications agrégées
- fidelity: Wireframes pour explorer les idées
- variations: Non, une seule proposition aboutie
- tweaks: Décide pour moi
- tone: Décide pour moi
- platform: Android
